Merged from:
- windows\
- jetson\

Merge rules used:
- Windows tree was used as the base because it contains the current Windows startup, mapping, and test fixes.
- Jetson-only files were then added on top.
- Shared code files were byte-identical across both trees except for config.local.json.
- Windows config.local.json was kept as merged\config.local.json.
- Jetson config.local.json was preserved as merged\config.ubuntu.local.json.
- Windows explicit config was preserved as merged\config.windows.local.json.
- Jetson backup files (*.bak*) were not copied into merged because they are not active project files.

Important:
- The original source snapshots were used for reference during the merge.
- The configs and one-off helper script in merged\ have been scrubbed for credentials and auth tokens.
- Review machine-specific paths and placeholder values before using this tree on a real host.
